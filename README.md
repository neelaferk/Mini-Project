- ## 📊 Project Overview
This Project presents an interactive Superstore Sales Analysis Dashboard developed using Power BI. The dashboard provides a comprehensive view of key business metrics, including total sales, profit, profit margin, quantity sold, category-wise performance, regional sales distribution, and sales trends over time."
